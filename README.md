# parking_management_project

This project is a Parking Management System built using FastAPI and SQLAlchemy.
It helps manage parking spaces by tracking available and occupied spots across multiple parking levels.

The system allows users to check parking availability, view empty spots by level, update vehicle entry or exit status, and monitor whether the parking area has reached full capacity.
It also includes a simple frontend interface served using static files.

Tech Stack

Python
FastAPI
SQLAlchemy
PostgreSQL / Database
HTML, CSS (for basic frontend)
